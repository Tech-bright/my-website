Web development

Web development is the work involved in the creating or developing a website and web application for internet or web . The website can also be seen as a place of marketing ,in the sense that ,the browser understand a certain language in which if communicated can be understood.
However,one of those languages is HTML(Hyper Text Markup Language). The ability for one to access the is through internet connection between the client and sever. The client is the user interconnected device such as computer connected to wifi phone connected to mobile network. Server on the other hand ,is a computer program or device that provides a service to another computer program and its user ,also known as client.
